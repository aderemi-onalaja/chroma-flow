# Platform & Tools (iOS First)

## Primary Recommendation: Unity (URP)
Best choice for achieving premium ripple physics and visual polish.

### Why Unity
- Excellent shader pipeline (URP)
- Strong iOS export support
- Mature audio, haptics, and touch handling
- Proven for mobile games with advanced visual effects

---

## Runner-Up: Godot 4
Strong open-source alternative.

### Pros
- Excellent 2D + shader support
- Lightweight
- No licensing cost

### Cons
- Smaller ecosystem
- Slightly more manual polish work for high-end effects

---

## Tooling Philosophy
- Core engine handles rendering + physics
- Levels defined as data (JSON / ScriptableObjects)
- No heavy UI frameworks
